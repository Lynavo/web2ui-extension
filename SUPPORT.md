# Support

Web2UI is a community-maintained, local-only extension. The project does not operate a hosted
capture service and does not provide private capture recovery or guaranteed response times.

## Before opening an issue

1. Confirm Chrome 106 or newer is in use.
2. Reproduce with the latest published release.
3. Try a normal HTTP(S) page; browser-internal and managed pages may block extension injection.
4. Remove credentials, cookies, personal data, and private page content.
5. Prefer a minimal synthetic HTML fixture that can be redistributed under the repository license.

Use the repository bug-report form for reproducible defects and the feature-request form for scoped
proposals. General Figma usage, browser administration, private website debugging, and custom
commercial integrations are outside this repository's support scope.
